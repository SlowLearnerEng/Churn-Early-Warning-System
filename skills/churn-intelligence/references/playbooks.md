## Sales Talk Tracks

### Response Decline

Use when response rate or response latency is the top driver.

1. Open with missed opportunity, not blame.
2. Show number of leads received and handled.
3. Compare fast-response sellers in the same category if benchmark evidence is available.
4. Offer quick replies, WhatsApp workflow, or sales ops support.

Suggested opener:

```text
Your account is still receiving buyer interest, but recent response behavior is lower than your normal baseline. If we fix lead handling before renewal, you can recover more of the value you already paid for.
```

### Low ROI Perception

Use when premium usage is weak near renewal.

1. Show qualified leads, repeat buyers, and category demand.
2. Translate activity into potential order value.
3. Recommend usage improvements before offering discounts.
4. Use discount only when value recovery and renewal risk justify it.

### Catalog Staleness

Use when product count, images, or category tags are stale.

1. Identify top products and missing attributes.
2. Recommend refreshing top 10-20 products first.
3. Pair catalog action with demand evidence.
4. Re-score seller seven days after update.

### Buyer Reactivation

Use when buyer inquiry frequency declines.

1. Segment by historical category interest.
2. Send category-specific nudge or saved-seller reminder.
3. Avoid over-contacting seasonal buyers outside expected buying window.

## Priority Rules

| Condition | Priority Adjustment |
| --- | --- |
| Renewal in 0-30 days and high value | escalate to manager. |
| High risk but low confidence | request human review before discount. |
| Critical risk and negative sentiment | support escalation before renewal pitch. |
| High recoverability and high value | schedule call within 72 hours. |
| Repeated unresolved alerts | suppress duplicates and create manager task. |

## Reasons To Stay Evidence

Use only metrics present in context:

| Evidence | Example Sentence |
| --- | --- |
| Qualified leads | "You generated 420 qualified leads in the last 6 months." |
| Repeat buyers | "67 leads came from repeat buyers, showing relationship value." |
| Category demand | "Demand in your category is up 18% in your city this quarter." |
| Response benchmark | "Faster responders in this category convert at a higher rate." |
| Catalog opportunity | "Refreshing top products can improve discovery before renewal." |
